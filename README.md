# 🚀 AI Project Architect

AI Project Architect is a full-stack MERN application that generates complete software project blueprints using AI.

Users can describe a project idea and instantly receive:

- System Architecture Diagram
- Database Schema
- API Endpoints
- Folder Structure
- Tech Stack Recommendations
- Development Roadmap
- Future Enhancements
- System Design Documentation

---

🔗 https://ai-project-architect-nine.vercel.app/
